# Project Context Templates

This directory contains templates for context files.

## Files

- `project.md` - Project overview
- `architecture.md` - Architecture documentation
- `conventions.md` - Coding conventions
- `current-state.md` - Current project state
- `decisions.md` - Technical decisions
- `task-history.md` - Task history
- `handoff.md` - Session handoff
- `agent-policy.md` - Agent policy rules

See packages/core/src/templates/template-types.ts for full implementations.
