# KontextMind Phase-by-Phase Agent Prompts

This zip contains separate implementation prompts for building **KontextMind** phase by phase.

Recommended usage:

1. Give `00_MASTER_PRODUCT_CONTEXT.md` to your coding agent first.
2. Then give `01_PHASE_1_FOUNDATION_INIT.md`.
3. After Phase 1 is working and tests pass, give Phase 2.
4. Continue phase by phase.

Do not ask the agent to implement all phases at once. KontextMind is a large tool, so phase-by-phase implementation will produce better code and fewer broken files.
